## Current Prototype

### Ascend Indiana WBL Employer Readiness Assessment

A lightweight employer-readiness assessment prototype that helps employers and intermediaries identify which work-based learning experiences best fit an organization’s current capacity, constraints, and talent goals.

Built in roughly 2–3 hours as an AI-assisted HTML, CSS, and JavaScript prototype, this concept turns a complex WBL decision framework into a tangible, testable user experience.

The prototype explores:
- Guided intake for employers exploring work-based learning options
- Conditional question logic based on employer capacity and constraints
- Scoring logic that maps responses to different WBL experience types
- Best-fit and stretch recommendations based on readiness signals
- Intermediary-friendly conversation support
- Dynamic next-step checklists tied to recommended experience types
- Lightweight print/export behavior for planning conversations

View prototype: https://scottrichardevans.github.io/ai-product-prototypes/ascendindiana/wbl/
