# AI Product Prototypes

A growing collection of AI-assisted product prototypes and workflow experiments.

These are lightweight concepts built to explore product ideas, clarify requirements, and test user experiences before engineering investment.

## Current Prototype

### Ascend Indiana WBL Employer Readiness Assessment

A lightweight employer-readiness assessment prototype that helps employers and intermediaries identify which work-based learning experiences best fit an organization’s current capacity, constraints, and talent goals.

Built in roughly 2–3 hours as an AI-assisted HTML, CSS, and JavaScript prototype, this concept turns a complex WBL decision framework into a tangible, testable user experience.

The prototype explores:

- Guided intake for employers and intermediaries exploring work-based learning options
- Dynamic scoring logic that recommends WBL experience types based on readiness, capacity, and constraints
- Conditional question paths that keep the assessment relevant to each organization
- Best-fit and stretch recommendations to support both immediate action and longer-term planning
- Next-step checklists for employers exploring how to move forward
- Intermediary-friendly support for consultative employer conversations
